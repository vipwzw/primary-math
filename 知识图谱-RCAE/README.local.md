## 15 种关系类型（节选最重要的 7 个）

| 关系 | 方向 | 含义 | 例 |
|---|---|---|---|
| `Prerequisite` | 有向 | A 是理解 B 的**先决条件**（学不懂 A 就学不懂 B） | 加法 → 乘法 |
| `Includes` / `IncludedIn` | 有向 | A **包含** B（A 是更广的概念） | 几何图形 ⊃ 三角形 |
| `AdvancesTo` | 有向 | A 是 B 的**深入或拓展** | 基础代数 → 高等代数 |
| `RelatedTo` | 无向 | A 与 B **相关** | 概率 ↔ 统计 |
| `AppliedIn` | 有向 | A **应用于** B | 几何 → 建筑设计 |
| `ContrastsWith` | 无向 | A 与 B **对比/差异** | 离散 ↔ 连续 |
| `SpecialCaseOf` | 有向 | A 是 B 的**特例** | 正方形 → 矩形 |

完整 12 种 + 系统中实际出现的额外 3 种（`DoesNotInclude` / `Excludes` / `SubsetOf`）见上游 `README.md`。

## 喂给 NotebookLM 的最佳实践

### 步骤
1. 在已经装了 12 本人教版 PDF 的同一个 NotebookLM notebook 里
2. **再把 `人教版-小学数学-知识图谱.json` 上传进去**（仅 2.9 MB，1 个 source）
3. 最好不要传全量版，否则 AI 会把北师大版概念也算进来

### 推荐的"知识图谱专用"提问模板

把这些 prompt 加到上一轮 48 小时学习法之上，效果显著放大：

**Q1 — 找根因**
```
我家孩子在"X 知识点（如：异分母分数加减法）"上反复出错。
基于人教版-小学数学-知识图谱.json 中这个节点的全部 Prerequisite
关系（递归向上 3 层），列出所有可能没掌握牢的"前置知识点"，
按"最可能是根因→最不可能"排序，并解释判断依据。
```

**Q2 — 排学习路径**
```
请基于知识图谱中的 Prerequisite 边，为"五年级下册"全部 254 个节点
生成一条最优学习顺序（拓扑排序），相邻可学的同时给出原因。
```

**Q3 — 找枢纽节点**
```
在 1632 个人教版节点中，按"被多少其他节点指向（即被多少后续知识依赖）"
排序，列出 Top 20 枢纽节点。这些是必须打磨的"高杠杆知识点"。
```

**Q4 — 跨年级知识链**
```
请追踪"分数"这一概念在人教版图谱中的完整演化链：
从最早出现的年级一直到六年级，按 AdvancesTo / Includes 关系串成
一条主链，标出每一站新增的认知负荷是什么。
```

**Q5 — 找断层**
```
基于图谱关系，请找出 5 处"教材中相邻两个知识点之间存在
认知跳跃过大"的情况（即 Prerequisite 链很短但难度跨度很大），
这些是需要家长/老师额外搭脚手架的地方。
```

## 重新生成人教版子集

如果你修改了原始 JSON 或上游数据更新了，重新跑：

```bash
python3 scripts/extract_pep_math_graph.py
```

只依赖标准库 `json` + `pathlib`，无需 pip install。
