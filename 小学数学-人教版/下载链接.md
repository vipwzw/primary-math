## 命令行批量下载

### Mac / Linux

```bash
# 方法 1：用 wget 按列表批量
wget -i https://raw.githubusercontent.com/vipwzw/primary-math/cursor/download-pep-math-textbooks-4ae7/小学数学-人教版/下载链接.txt

# 方法 2：用 aria2c 多线程并行（推荐，更快）
curl -O https://raw.githubusercontent.com/vipwzw/primary-math/cursor/download-pep-math-textbooks-4ae7/小学数学-人教版/下载链接.txt
aria2c -i 下载链接.txt -j 4 -x 4
```

  $url = "$base/$([uri]::EscapeDataString($f))"
  Invoke-WebRequest -Uri $url -OutFile $f
}
