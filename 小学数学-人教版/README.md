## 喂给 NotebookLM 的建议

NotebookLM 免费版允许最多 50 个 source、单文件 ≤ 200 MB，本套 12 个 PDF 完全在限额内。

**推荐两种组织方式：**

### 方案 A：一个 notebook 装全部 12 册（适合"全学科地图"提问）
直接把 12 个 PDF 全部上传到同一个 notebook，然后跑：
```
基于我上传的人教版小学数学 1–6 年级全部教材，请提炼出贯穿
全部内容的 5–7 个核心数学思想（mental models）……
```

### 方案 B：按学段分 3 个 notebook（适合"按年级精讲"）
- 低段 notebook：1–2 年级（4 册，约 147 MB）
- 中段 notebook：3–4 年级（4 册，约 81 MB）
- 高段 notebook：5–6 年级（4 册，约 70 MB）

中段、高段抽取的心智模型颗粒度更细，不会被低段图画式内容稀释。

## 获取方式

PDF 文件已经直接入库，最快的方式：

```bash
git clone https://github.com/vipwzw/primary-math.git
cd primary-math
ls 小学数学-人教版/    # 12 本 PDF 都在这
```

或者：直接在 GitHub 网页打开 `小学数学-人教版/` 目录，点任一 PDF 的 "Download raw file" 单独下载。

## 重新下载 / 校验

如果想验证完整性或上游更新了，运行仓库根目录的脚本：

```bash
bash scripts/download_pep_math.sh
```

脚本会跳过已下载且大小一致的文件，可放心多次运行。
